# TASK 1 - Web Scraping Project

## Project Title
Amazon Smartphone Data Web Scraping

## Objective
The objective of this project is to collect smartphone data from public web pages using web scraping tools and create a structured dataset for further analysis.

## Tools Used
- Octoparse
- Microsoft Excel

## Description
In this project, smartphone data was extracted from Amazon product pages using the Octoparse web scraping tool. The collected data was organized and stored in an Excel file for analysis purposes.

The dataset includes:
- Smartphone Name
- Price
- Ratings
- Number of Reviews
- Product Information

## Steps Performed
1. Selected Amazon smartphone listings as the data source.
2. Used Octoparse to extract product data from the website.
3. Collected relevant information such as prices, ratings, and product names.
4. Removed unnecessary column like image url, duplicate column, delivery columns and changed names to correct names for all. 
4. Organized the extracted data into a structured dataset.
5. Exported the final dataset into an Excel file.

## Output
The final dataset was successfully created and saved in Excel format.

## Learning Outcomes
- Learned the basics of web scraping
- Understood website data extraction
- Worked with automated scraping tools
- Created a custom dataset for analysis

## Conclusion
This project helped in understanding how data can be collected from websites using web scraping tools and prepared for future data analysis tasks.
